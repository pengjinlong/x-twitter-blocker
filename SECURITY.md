| Version | Supported |
| ------- | --------- |

## Reporting a Vulnerability

If you discover a security vulnerability, please do **NOT** open a public Issue.

Instead, please report it privately by emailing the maintainers or by using [GitHub's private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability).

We will respond within 48 hours and work with you to resolve the issue before any public disclosure.

## Privacy Note

This extension/script processes all data locally in your browser. No keywords, browsing data, or personal information is ever transmitted to any server.
